# 📱 レジ売上管理アプリ

iPhoneから手軽に売上を入力し、従業員ごとに担当を記録・集計できるモバイル対応ウェブアプリ。  
**管理者／スタッフの2段階権限**とPINログインを搭載。

## ✅ 実装済み機能

### 🔑 ログイン（login.html）
- 担当者をタップして選択
- 4桁のテンキーでPINを入力
- 認証成功で売上入力ページへ遷移
- 未ログイン時は自動でログイン画面へリダイレクト
- ヘッダーにログインユーザー名・権限バッジ・ログアウトボタンを表示

### 💰 売上入力（index.html）
- 従業員タブで担当者を切り替えて入力（全スタッフが全員分切替可）
- 支払い方法・カテゴリ・備考の入力
- **入力金額を小計（税抜）として自動計算：小計・消費税（10%）・税込合計を保存**
- 本日の売上合計（税込）リアルタイム表示
- 売上カードに「税込合計 / 小計 / 消費税」を表示
- 売上一覧を「全員/担当者別」タブで切り替え
- 削除は管理者のみ

### 📊 日次集計・レジ締め（summary.html）
- 日付選択で任意日の集計（**全スタッフがアクセス可**）
- **本日の売上合計を税込で表示、小計・消費税内訳を補足表示**
- 支払い方法別・**従業員別**売上集計（税込合計ベース）
- **レジ締め計算は税込現金売上で突き合わせ**・過不足判定・保存（**全スタッフが実行可**）

### 📋 売上履歴（history.html）※管理者専用
- 期間・支払方法・担当者フィルター
- 期間集計（従業員別ランキング付き）
- 日別グループ表示

### 👤 従業員管理（staff.html）※管理者専用
- 従業員の追加（名前・PIN・カラー・権限）
- 編集（名前・PIN・カラー・権限）
- 権限の昇格／降格ボタン（ワンタップで切替）
- 削除（論理削除）
- 権限説明カード（管理者／スタッフの違いを明示）
- **営業日の設定**（下記参照）

### 📅 営業日設定（staff.html ／ 管理者専用）
- 管理者が任意の日付を「営業日」として設定できる
- 設定した営業日が**全スタッフの売上入力日・日次集計の初期日付**に自動反映される
- 例：5月2日を営業日に設定 → 全スタッフが入力した売上は5月2日付で記録される
- 実際の日付（端末の今日）と異なる場合、売上入力・日次集計ページに**オレンジ色の警告バナー**が表示される
- 「設定を解除」ボタンで端末の実日付に戻すことができる
- 設定は `app_settings` テーブルに保存され、複数端末間で即時同期される

## 🗄 データモデル

### `employees` テーブル
| フィールド | 型 | 説明 |
|---|---|---|
| id | text | レコードID |
| name | text | 従業員名 |
| color | text | 表示カラー（hex） |
| is_active | bool | 有効フラグ |
| sort_order | number | 表示順 |
| role | text | `admin`（管理者）/ `staff`（スタッフ） |
| pin | text | 4桁PINコード（平文保存） |

### `sales` テーブル
| フィールド | 型 | 説明 |
|---|---|---|
| id | text | レコードID |
| date | text | 日付 YYYY-MM-DD |
| time | text | 時刻 HH:MM |
| amount | number | 小計金額（税抜）※ subtotal と同値 |
| subtotal | number | 小計金額（税抜） |
| tax | number | 消費税額（10%） |
| total | number | 税込合計金額 |
| payment_method | text | 現金 / クレジット / 電子マネー / QRコード |
| category | text | カテゴリ（任意） |
| note | text | 備考（任意） |
| staff_name | text | 担当従業員名 |

### `daily_register` テーブル
| フィールド | 型 | 説明 |
|---|---|---|
| id | text | レコードID |
| date | text | 日付 YYYY-MM-DD |
| cash_in_register | number | 実際に数えたレジ内現金 |
| opening_balance | number | 開始時レジ金 |
| note | text | 備考 |
| is_closed | bool | 締め済みフラグ |

### `app_settings` テーブル
| フィールド | 型 | 説明 |
|---|---|---|
| id | text | レコードID |
| key | text | 設定キー（例: `business_date`） |
| value | text | 設定値（例: `2025-05-02`） |
| updated_by | text | 最終更新者名 |
| description | text | 設定の説明 |
